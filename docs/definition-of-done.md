# Definition of Done (DoD) - Semana 9

Uma funcionalidade só é considerada PRONTA quando:
1. [x] O código está funcionando (testado manualmente)
2. [x] Segue o padrão MVC + Services
3. [x] Erros são tratados com classes customizadas
4. [x] Validações estão implementadas
5. [x] A rota está documentada no Swagger
6. [x] Pelo menos um outro membro revisou o código
7. [x] Commit feito com mensagem descritiva
8. [x] Card movido para "Concluído" no Kanban
