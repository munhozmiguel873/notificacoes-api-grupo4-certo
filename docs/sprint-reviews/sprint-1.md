## 💡 Lições aprendidas

* Definir claramente as rotas e o escopo antes do desenvolvimento acelera a implementação

* Documentar a API desde o início evita divergências entre implementação e especificação
* Planejar tempo para testes manuais e ajustes na documentação é essencial para uma entrega de qualidade
